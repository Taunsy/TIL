## django shit3

* write 페이지 -> 게시글 작성 -> create -> DB 저장 -> list 페이지 리다이렉트
* list/url -> id값 확인 -> read/{{ DB.id }}/ -> 해당 id에 해당하는 DB 호출 -> read/<num> 페이지
* python manage.py makemigrations: migrate 된 목록을 DB로 넘김
